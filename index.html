Grid is made for two-dimensional layout while Flexbox is for one.This means Flexbox can work on either row or columns at a time, but Grids can work on both. Flexbox, gives you more flexibility while working on either element (row or column).But grid will give you more option in both axis layout making. Grid used to use overall layout structuring while flexbox vastly used to centering and aligning inside elements.

The significant difference between Tailwind CSS and Bootstrap is that Tailwind offers predesigned widgets to build a site from scratch with fast UI development, while Bootstrap comes with a set of pre-styled responsive, mobile-first components that possess a definite UI kit. According to me, the bootstrap much more beginner friendly and the tailwind offers much more flexibity in the design. Both the frameworks has a variety of features and UI components as well.

In vanila CSS every element has a box model.In CSS, the term "box model" is used when talking about design and layout. CSS box model consists of content,padding,border and margin.

The tags that provide meaning to an HTML page rather than just presentation is called semantic tagSimply put which tags are much more meaningful and It makes HTML more comprehensible by better defining the different sections and layout of web pages.
